# Platform Specifications Reference

Technical specs and native content patterns for major platforms.

---

## X (Twitter)

### Feed Post

| Spec | Value |
|------|-------|
| Character limit | 280 (free) / 25,000 (Premium) |
| Image ratio | 16:9 (1200x675) or 1:1 (1080x1080) |
| Max images | 4 |
| Video ratio | 16:9, 1:1, or 9:16 |
| Video duration | 0:01–2:20 (free) / up to 4 hours (Premium) |
| Video resolution | 1280x720 min, 1920x1080 recommended |

**Native patterns:**
- Single punchy observation
- Hot take with proof
- Numbered list (1-5 items)
- Quote + reaction
- Before/after comparison

### Thread

| Spec | Value |
|------|-------|
| Max tweets | 25 (recommended 5-12 for engagement) |
| Per-tweet limit | 280 characters |
| Images | 1-4 per tweet |

**Native patterns:**
- Hook → Context → Main points → CTA
- Numbered breakdown (1/, 2/, 3/...)
- Story arc with cliffhangers between tweets
- Data thread: stat → insight → stat → insight
- First tweet hooks, last tweet CTAs

### Video (Native)

| Spec | Value |
|------|-------|
| Aspect ratios | 16:9, 1:1, 9:16 |
| Resolution | 720p min, 1080p recommended |
| Duration | 15 sec–2:20 min optimal |
| File size | Max 512MB |
| Captions | Recommended (85% watch without sound) |

**Native patterns:**
- Quick tip (15-30 sec)
- Breakdown/explainer (60-90 sec)
- Behind the scenes
- Reaction/response video

### Spaces

| Spec | Value |
|------|-------|
| Duration | Up to 4 hours |
| Speakers | Up to 13 |
| Recording | Available for hosts |

**Native patterns:**
- Q&A with community
- Industry discussion/panel
- Product announcement + live questions
- Weekly recurring format

## TikTok

| Spec | Value |
|------|-------|
| Aspect ratio | 9:16 (1080x1920) |
| Duration | 15 sec–10 min (15-60 sec optimal) |
| File size | Max 287MB |
| Resolution | 720p min, 1080p recommended |
| Safe area - top | 150px (username/follow button) |

- Before/after reveals

| Spec | Value |

| Duration | 15 sec max per story |
| Expires | 24 hours |

**Native patterns:**

## Brand Tone Adaptation by Channel

How brand voice typically adapts across platforms. Use this as a starting template, then customize for the specific brand.

| Voice Element | X | LinkedIn | TikTok | Instagram | Email | Blog |
|---------------|---|----------|--------|-----------|-------|------|
| **Formality** | Casual | Professional | Very casual | Casual-visual | Semi-formal | Semi-formal |
| **Humor** | Encouraged | Subtle | Expected | Visual humor | Light | Depends on brand |
| **Technical depth** | Low-medium | Medium-high | Low | Low | Medium | High |
| **Emoji use** | Moderate | Minimal | Heavy | Heavy | Minimal | None |
| **Sentence length** | Short, punchy | Medium | Ultra-short | Short captions | Varied | Varied |
| **CTA style** | Soft (reply, RT) | Direct (link, DM) | Soft (follow, comment) | Link in bio | Direct (button) | Direct (inline) |

**Adaptation principles:**
- Voice (personality) stays consistent across all channels
- Tone (formality, humor, depth) adapts to platform norms
- Quick check for each piece: Does this sound like us? Does it fit the channel? Would our audience talk this way here?
